# Fun and Balance Rubric (Pinvestor)

Score each axis from 1-5 and explain the score.

## 1) Rule-Bending Strength
- Does the element modify a gameplay rule (not only output numbers)?
- Does it create a new decision pattern?

## 2) Behavior Identity
- Is the behavior memorable and explainable in one sentence?
- Would players recognize it as unique in a run recap?

## 3) Synergy Breadth
- Does it connect to multiple companies/archetypes?
- Does it support at least one surprising combo?

## 4) Counterplay and Boundaries
- Are there clear constraints/costs/timing windows?
- Can it fail or underperform in some board states?

## 5) Replay Value
- Does it create varied outcomes across runs?
- Does it encourage adaptation to run theme/market conditions?

## 6) Implementation Feasibility
- Can this fit current architecture (services/resolvers/event flow) without hacks?
- Are triggers and precedence rules explicit enough to implement safely?

## Red Flags (Reject or Redesign)
- Pure stat inflation with no new behavior.
- No meaningful synergy/counter-synergy mapping.
- Always-correct pick regardless of board state.
- Infinite loops or low-interaction autopilot patterns.
