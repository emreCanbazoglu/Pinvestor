# Humor Style Guide (Absurd Comedy)

Target tone: chaotic, irreverent, high-contrast absurdity with clever game logic underneath.

Use this as tonal inspiration:
- Fast escalation from normal setup to ridiculous consequence.
- Corporate satire, startup nonsense, cursed tech optimism.
- Silly names with clear gameplay meaning.
- Punchy, memorable flavor text.

Do not imitate specific protected characters, quotes, or scenes.
Do not copy trademark catchphrases or direct wording from existing IP.

## Humor Design Rules

1. Keep the joke mechanical.
- The humor must emerge from behavior, not only text.
- If the mechanic is removed and only name/text stays, redesign.

2. Keep readability first.
- Players should parse trigger and effect quickly.
- One clear comedic premise per element; avoid stacked random gags.

3. Mix absurd with strategic clarity.
- The effect can be weird, but counters and limits must be explicit.
- A funny card still needs clear board-state use cases.

4. Prefer contrast.
- Pair high-stakes systems with ridiculous framing.
- Example pattern: serious economic pressure + nonsense mechanism.

5. Make outcomes story-worthy.
- Aim for "I cannot believe that worked" moments.
- Support emergent chain reactions players retell after runs.

## Naming Pattern Suggestions

- Fake startup names: "Tax Evasion-as-a-Service", "Mood-Swing Metrics", "Quantum Nepotism"
- Booster names: "Liability Wizard", "Hostile Memeover", "Regulatory Loophole Gun"
- Event names: "Bull Market Possession", "Audit of the Damned", "Venture Capital Eclipse"

## Flavor Text Constraints

- Keep to one line where possible.
- Avoid references requiring external IP knowledge.
- Punchline should reinforce effect identity, not be random.
